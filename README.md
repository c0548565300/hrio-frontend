# HR.io - Candidate Management Dashboard 🚀

A modern, glassmorphism-inspired HR dashboard for analyzing and managing GitHub candidate profiles. 

## Tech Stack
* **Frontend Framework:** React (Vite)
* **UI Component Library:** Chakra UI v3
* **Animations:** Framer Motion
* **Database & Auth:** Firebase
* **Data Fetching:** React Query (TanStack)

## Design System
Built with a custom "Neon-Glass" design system, featuring dynamic mesh gradients, strict type tokens, and modern recipes.
